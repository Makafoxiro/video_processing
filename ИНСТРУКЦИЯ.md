# TikTok Downloader — Инструкция по запуску

---

## 1. Требования

| Что нужно | Где взять |
|---|---|
| **Python 3.9–3.14** | https://python.org/downloads → при установке поставь галочку «Add Python to PATH» |
| **yt-dlp** | устанавливается командой ниже |
| **gallery-dl** | устанавливается командой ниже |

---

## 2. Установка зависимостей

Открой **командную строку** (`Win + R` → `cmd`) и выполни:

```
py -m pip install yt-dlp gallery-dl
```

> Если команда `py` не найдена — попробуй `python` вместо `py`.

---

## 3. Расположение файлов

Убедись, что файл `app.py` лежит в папке:

```
C:\Users\<твой_пользователь>\Desktop\video_processing\
```

Папки для загрузок создаются автоматически при первом запуске, но можно создать вручную заранее:

```
C:\video_processing\favorites\
C:\video_processing\tiktok_photos\
```

---

## 4. Запуск

```
cd C:\Users\<твой_пользователь>\Desktop\video_processing
py app.py
```

Или просто **дважды кликни** по `app.py` (если Python ассоциирован с `.py` файлами).

---

## 5. Первоначальная настройка в приложении

После запуска перейди на вкладку **⚙️ Настройки**:

1. **Никнейм TikTok** — твой аккаунт (без `@`)
2. **Папка для избранного / видео / фото** — укажи куда сохранять
3. **Cookies** — нужны для скачивания приватных/избранных видео. Есть два варианта:
   - **Галочка «Использовать браузер»** — приложение само возьмёт cookies из Chrome (браузер должен быть закрыт)
   - **Путь к cookies.txt** — экспортируй вручную расширением:
     - Chrome: `Get cookies.txt LOCALLY`
     - Firefox: `cookies.txt`

Нажми **Сохранить** после настройки.

---

## 6. Проверка зависимостей

На вкладке **⚙️ Настройки** внизу есть блок с зелёными/красными кружочками:

- 🟢 **yt-dlp установлен** — всё ок
- 🟢 **gallery-dl установлен** — всё ок
- 🔴 красный — запусти команду из подсказки рядом

---

## 7. Использование

### ⭐ Избранное
Скачивает все видео из раздела «Избранное» твоего аккаунта.
1. Введи никнейм
2. Нажми **▶ СТАРТ**

### 👤 По аккаунтам
Скачивает все видео с указанных аккаунтов (список задаётся в настройках).

### 🖼 Фото
Скачивает фотокарусели через `gallery-dl`.

---

## 8. Возможные ошибки

| Ошибка | Решение |
|---|---|
| `py не является внутренней командой` | Переустанови Python с галочкой «Add to PATH» |
| `No module named yt_dlp` | Выполни `py -m pip install yt-dlp` |
| Видео не скачивается | Обнови yt-dlp: `py -m pip install -U yt-dlp` |
| Ошибка cookies | Экспортируй cookies заново — они протухают |
